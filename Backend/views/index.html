Route Description

Method Path Description
01 get / Takes you to the home page (index.html)
02 post /users/login login account
03 get /users returns users array
04 get /users/user:ID returns object of user with matching userID
05 put /users/user:ID updates user details with matching ID
06 post /register encrypts password user inserts, saves user successfully in relevant database, returns JSON web token to frontend.
07 delete /users/user:ID removes user from database.
08 delete /products shows all products from database.
09 delete /products/product:id shows a product from database with matching ID.
